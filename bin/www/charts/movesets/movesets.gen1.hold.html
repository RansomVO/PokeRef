Pokémon Move Sets - Gen 1

NOTE: Last Updated 15-Feb-2018

This chart lists the Move Sets for Pokémon.
Just look up the Pokémon you are interested in, find the entry for the Move Set for your Pokémon, and compare it to the other possible Move Sets.

Here are the things you will find:

  • Move Set: This is a possible combination of a Fast Attack and a Charged Attack.
  • STAB: (Same Type Attack Bonus) Tells whether the Pokémon get a 25% bonus because the Move's Type is the same as the Pokémon's type.
  • Move Set DPS: (Damage Per Second) Tells the raw DPS for Move Set itself, without taking into account the Pokémon's Stats.
  • True DPS: Tells the DPS for Move Set, for the specified Pokémon.
  • %: A comparison of the Move Set's True DPS, to the True DPS of the Pokémon's best possible Move Set that doesn't include a Legacy Move.

For a technical description of the formulas used, check out the Move Set Formulas.

Example:
  • +pidgey: All Pokémon in the Pidgey family.

  • 1-10,B: All Pokémon with:

    a name that begins with B.

&
Specifies one criteria AND another.
Example:
  • 1-10&B: All Pokémon with:
    IDs between 1 and 10
    - AND -
    a name that begins with B.
